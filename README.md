# cp-ROIs-Cordillera-Escalera
Pipeline y recursos para la construcción de una base de datos de genomas cloroplastidiales para Nanopore Adaptive Sampling (NAS). Incluye recuperación de genomas, filtrado de contaminantes/repetitivos, selección de ROIs, depuración de alineamientos y generación de archivos .bed y .fasta para regiones objetivo.
